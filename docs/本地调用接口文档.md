# 本地 SDK 调用接口文档

本机（`D:\dev\Project\StockSDK`）已部署两个股票数据 SDK，均为 Python 3.11.4 x64，各自独立 venv。
本文档说明**在本机直接用 Python 调用**的方式。凭据走 `.env`，不要硬编码。

> **推荐：用封装好的简化模块（见下方"封装模块"一节）。** 调用时无需写登录/令牌/账密，一行取数返回 DataFrame。
> 下面的"原生 SDK 调用"是底层细节，需要更细控制时再用。

---

## 封装模块（推荐，调用最简）

模块内部自动处理登录（读 `.env` / 用本机令牌）、复用连接、统一错误、默认返回 `pandas.DataFrame`。
**调用时不传任何账密/令牌**。

### EM（`.venv-em` 下，`import em_api as em`）
```python
import em_api as em
df = em.csd("300059.SZ", "CLOSE", "2024-01-01", "2024-01-05")   # 序列
df = em.css("300059.SZ,000002.SZ", "OPEN,CLOSE", "TradeDate=20240105")  # 截面
df = em.cses("blockcode", "OPEN,CLOSE")                          # 板块截面
df = em.tradedates("2024-01-01", "2024-01-31")                  # 交易日历
# 需要原始对象：传 as_df=False
```
返回 DataFrame：行=代码×日期，列=指标。

### iFinD（`.venv-ths` 下，`import ths_api as ths`）
```python
import ths_api as ths
df = ths.history("300033.SZ", "open;high;low;close", "2024-01-01", "2024-01-05")  # 历史行情
df = ths.basic("300033.SZ", "ths_stock_short_name_stock", "")    # 基础/截面
df = ths.realtime("300033.SZ", "open;high;now", "")             # 实时
# 需要原始 dict：传 as_df=False
```

说明：
- 首次调用自动登录，之后复用；进程退出时自动登出（也可 `em.logout()` / `ths.logout()` 手动）。
- 出错抛 `RuntimeError`（含错误码），不用自己判 ErrorCode。
- 取别的指标：用 SuperCommand / 官网超级命令生成指标名，填进对应参数即可。

## 一、同花顺 iFinD（`.venv-ths`）

### 启动
```bat
.venv-ths\Scripts\activate
```
登录方式：账号密码（任意设备通用，单点登录）。

### 登录 / 登出
```python
from iFinDPy import THS_iFinDLogin, THS_iFinDLogout
code = THS_iFinDLogin("touzi228", "<密码>")   # 返回 0 = 成功
# ... 取数 ...
THS_iFinDLogout()
```

### 常用取数函数

| 函数 | 用途 | 签名示例 |
|---|---|---|
| `THS_HistoryQuotes` | 历史行情序列 | `(codes, indicators, params, begin, end)` |
| `THS_RealtimeQuotes` | 实时行情 | `(codes, indicators, params)` |
| `THS_BasicData` | 基础/截面数据 | `(codes, indicators, params)` |
| `THS_DateSequence` | 日期序列指标 | `(codes, indicators, params, begin, end)` |
| `THS_DataPool` | 数据池（板块成分等） | `(poolname, params, indicators)` |
| `THS_EDB` | 宏观经济数据 | `(indicators, begin, end)` |
| `THS_DateOffset` | 交易日偏移 | `(exchange, params)` |

### 取数示例
```python
from iFinDPy import THS_HistoryQuotes
r = THS_HistoryQuotes(
    "300033.SZ",                 # 代码，多个用逗号
    "open;high;low;close",       # 指标，多个用分号
    "",                          # 附加参数
    "2024-01-01", "2024-01-05",  # 起止日期 YYYY-MM-DD
)
# 返回 dict：r['errorcode']==0 成功；
# r['tables'][0]['time'] 是日期列表，['table']['close'] 是收盘价列表
```

### 返回结构（dict）
```python
{
  'errorcode': 0,
  'errmsg': '',
  'tables': [{'thscode': '300033.SZ',
              'time': ['2024-01-02', ...],
              'table': {'open': [...], 'close': [...]}}],
}
```

### 注意
- **单点登录**：同一账号同一时刻只能一处登录，多处会互相挤掉。
- 指标代码可用 `SuperCommand` 工具或官网超级命令生成。

## 二、东方财富 EMQuantAPI（`.venv-em`）

### 启动
```bat
.venv-em\Scripts\activate
```
登录方式：本机已激活 `userInfo` 令牌，`c.start()` 免账密。

### 登录 / 登出
```python
from EmQuantAPI import c
r = c.start()                 # 令牌登录，r.ErrorCode==0 成功
# 可选账密登录（不生成令牌）：c.start("UserName=账号,Password=密码")
# ... 取数 ...
c.stop()
```

### 常用取数函数

| 函数 | 用途 | 签名 |
|---|---|---|
| `c.css` | 截面数据（多代码某日多指标） | `css(codes, indicators, options)` |
| `c.csd` | 序列数据（单/多代码时间区间） | `csd(codes, indicators, startdate, enddate, options)` |
| `c.cses` | 板块截面 | `cses(blockcodes, indicators, options)` |
| `c.sector` | 板块成分 | `sector(pukeycode, enddate, options)` |
| `c.tradedates` | 交易日历 | `tradedates(startdate, enddate, options)` |
| `c.getdate` | 交易日偏移 | `getdate(tradedate, offday, options)` |
| `c.edb` | 宏观数据 | `edb(codes, options)` |

### 取数示例
```python
from EmQuantAPI import c
c.start()
# 序列：300059.SZ 收盘价
d = c.csd("300059.SZ", "CLOSE", "2024-01-01", "2024-01-05", "")
# d.ErrorCode==0；d.Data = {'300059.SZ': [[13.8, 13.71, 13.46, 13.28]]}

# 截面：多代码某日开盘/收盘
s = c.css("300059.SZ,000002.SZ", "OPEN,CLOSE", "TradeDate=20240105,Ispandas=0")
c.stop()
```

### 返回结构（EmQuantData 对象）
```python
d.ErrorCode    # 0 = 成功
d.ErrorMsg     # 'success'
d.Codes        # ['300059.SZ']  代码列表
d.Indicators   # ['CLOSE']      指标列表
d.Dates        # 日期列表
d.Data         # {'300059.SZ': [[...]]}  数据，按 [指标][日期] 排列
```

### 限频与流量（重要）
- 截面/序列函数：**每分钟请求数 ≤ 700 次**。
- 按品种有**流量额度**，用完该品种取不到数：官网 `https://quantapi.eastmoney.com/Flow` 查询。
- 多设备：令牌绑当前设备；`ForceLogin=1` 可抢登，多并发需开通多点权限。

### 常见错误码
| 错误码 | 含义 | 处理 |
|---|---|---|
| 10001002 | 账密错误 | 核对账密 |
| 10001003/04 | 无 API 权限/过期 | 联系客户经理 |
| 10001009 | 登录数超限 | 加 `ForceLogin=1` 或开多点 |
| 10001019 | 激活设备与登录设备不一致 | 换设备需重新激活 |
| 10001020 | userInfo 失效 | 改密后重新激活 |
