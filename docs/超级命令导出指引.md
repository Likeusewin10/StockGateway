# 超级命令 / 数据浏览器 导出指引（Track B）

> 目的：把两家桌面客户端的**全量字段级指标库**导出成 Excel，
> 交给我归一化合并进 `docs/catalog/`。这是 API 层拿不到、只能 GUI 导出的部分。

## 前置

- 已安装并登录两个桌面客户端（账密见 `.env`，勿与 SDK 同账号同时单点登录 iFinD）。
- 新建空目录 `docs/catalog/_raw/`，导出文件都丢这里。

## 一、东方财富 Choice 数据终端

下载：https://choice.eastmoney.com/download
默认安装路径：`D:\Program Files (x86)\Eastmoney\Choice\ChoiceLoader.exe`

### 导出指标库（按维度）
1. 打开 Choice → 顶部菜单找 **「数据浏览器」**（或「数据」→「数据浏览器」）。
2. 左侧指标树按维度展开：**行情 / 财务报表 / 财务分析 / 估值 / 一致预期 / 技术指标 / 公司治理**。
3. 选中某维度全部指标 → 工具栏 **「导出」→ Excel**。
4. 文件命名：`em_<维度>.xlsx`，如 `em_财务分析.xlsx`，放入 `docs/catalog/_raw/`。

### 导出 EDB 宏观指标树
1. 顶部 **「EDB」**（经济数据库）。
2. 左侧分类树（宏观/行业/区域）逐类展开 → 「导出」目录与指标 ID。
3. 命名：`em_edb_<分类>.xlsx`。

### 提示
- 指标代码列是关键（css/csd 直接用）。务必导出"指标代码 + 中文名 + 单位/说明"。
- 单次导出可能有条数上限，按维度分批即可。

## 二、同花顺 iFinD 终端

下载：官网 https://www.51ifind.com → 下载中心。

### 导出指标库（数据接口 / 超级命令）
1. 打开 iFinD → 找 **「数据接口」** 或 **「超级命令」**（不同版本菜单名略异，通常在「工具」或顶部）。
2. 指标库按维度展开：**行情 / 财务 / 估值 / 盈利预测 / 技术 / 公司资料 / 行业**。
3. 选维度 → **「导出」**（Excel/CSV）。
4. 命名：`ths_<维度>.xlsx`，放入 `docs/catalog/_raw/`。

### 导出 EDB / 宏观
1. 「EDB」或「宏观数据」浏览器 → 分类树展开 → 导出指标 ID + 名称。
2. 命名：`ths_edb_<分类>.xlsx`。

### 提示
- iFinD 指标代码形如 `ths_xxx_stock`；EDB 形如 `M001620326`。两列都要。
- 单点登录：导出期间若 SDK 服务在跑且用同账号，会互相挤掉，错峰操作。
