# 同花顺 iFinD 指标字段手册

> 数据来源:iFinD DeepSeek MCP 问答式取数接口(`mcp__hexin-ifind-ds-*` 系列)。
> 字段元数据从每次问答返回的 `# 指标参数信息` JSON(或返回表头)中提取,去重后按维度归并。
> 因 MCP 为问答式接口,无法穷举某维度绝对全集,本手册覆盖各维度主流字段。

- **维度数**:20 个子维度 / 6 大类
- **字段总数**:502 个(已按字段名去重)
- **代表证券**:A股=贵州茅台600519.SH;债券=24附息国债25/华海转债;基金=易方达蓝筹精选005827;指数=沪深300;板块=申万食品饮料/白酒;港美股=苹果AAPL.O

**说明**:参数列保留有区分度的调用参数(如 `报表类型`/`复权方式`/技术指标的 `指标选项`/`周期`),省略每字段通用的 `报告期`/`交易日期`/`单位`。技术指标的 `指标选项=X|Y|Z` 表示同一规范名通过该参数切换输出分量。
